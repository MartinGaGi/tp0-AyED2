## 📚 Descripción
Este proyecto consiste en el desarrollo de un sistema académico básico en Java, utilizando Programación Orientada a Objetos (POO).

El sistema permite gestionar estudiantes, materias y carreras, aplicando conceptos como encapsulamiento, relaciones entre clases y validaciones.

---

## 🧩 Clases implementadas

- **Estudiante**
  - Datos personales
  - Lista de materias
  - Cálculo de promedio

- **Materia**
  - Nombre, código, créditos y calificación

- **Carrera**
  - Gestión de estudiantes
  - Búsqueda y listado

- **Universidad**
  - Contenedor de estudiantes

- **App**
  - Menú interactivo por consola

---

## ⚙️ Funcionalidades

- ✔ Crear estudiantes  
- ✔ Agregar materias a estudiantes  
- ✔ Calcular promedio de calificaciones  
- ✔ Listar estudiantes  
- ✔ Buscar estudiantes  
- ✔ Menú interactivo  

---

## 🔐 Conceptos aplicados

- Encapsulamiento (atributos privados + getters/setters)
- Uso de constructores
- Relaciones entre clases
- Uso de ArrayList
- Validaciones de datos

---

## ▶️ Cómo ejecutar

1. Compilar las clases
2. Ejecutar `App.java`
3. Usar el menú desde consola
